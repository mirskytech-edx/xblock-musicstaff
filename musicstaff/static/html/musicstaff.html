Quick start

  • "K:" followed by a upper or lower case letter specifies the key.
  • "M:" specifies the meter in fraction notation. eg "M:6/8"
  • "L:" specifies the default length of a note. eg "L:1/4" has each note being a quarter-note
  • Each note is written as a letter, separated by a space; upper case to denote the bottom octave and lower case letters for the top octave.
  • To go down an octave, just put a comma after the letter and to go up an octave use an apostrophe.
  • If a letter is followed by a number, it is used to multiply its length eg. A2 is a half-note if the default is L:1/4
  • If a letter is followed by a slash and a number, it is used to divide its length eg. A/2 is an eitghth-note if the default is L:1/4
  • Example: 
    M:C
L:1/4
K:C
